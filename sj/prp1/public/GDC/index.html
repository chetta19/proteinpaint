GDC API usage

/ssms Query list of ssm by isoform/gene/case id

/ssms Get canonical isoform using an SSM id

/ssm_occurrences Query list of cases harboring ssm for a gene/isoform

/ssm_occurrences Query ssm read depth

/cases Batch convertion of aliquot IDs to submitter IDs

/files Get list of open-access projects (otherwise are controlled)

/files and /cases Query eligible BAM files by a case/file name

/analysis/top_mutated_genes_by_project Top genes (CGC or not) with most SSM from a cohort

/data/<fileUUID> To check if a user has access to the BAM file (by passing token in request header)

/slicing/view/<fileUUID>?region=chr:start-stop Example To slice a BAM file at given position (by passing token in request header)

(graphql) Get number of cases breakdown by variables

(not in use) Convert a batch of aliquot IDs to submitter IDs
